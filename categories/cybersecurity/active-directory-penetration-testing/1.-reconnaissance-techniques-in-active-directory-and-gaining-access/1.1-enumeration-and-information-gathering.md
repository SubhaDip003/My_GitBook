# 1.1 Enumeration and Information Gathering

**Enumeration and Information Gathering** are the first and most important steps in attacking an Active Directory environment. In this phase, the goal is to collect as much information as possible about the target network, such as users, groups, computers, services, and permissions.

This process helps in understanding how the network is structured and where potential weaknesses may exist. By using different techniques and tools, attackers or security professionals can identify valuable details that are not immediately visible.

The information gathered during enumeration is then used to find vulnerabilities and misconfigurations. This allows an attacker to plan and execute further attacks more effectively, making enumeration a critical step in any successful Active Directory exploitation process.
